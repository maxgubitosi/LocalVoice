LocalVoice — Áreas de mejora

¿Por dónde empezamos? Seleccioná las que querés abordar primero.

1

Bug: Error de Sparkle al iniciar

Al hacer make run, aparece un cartel de error relacionado con actualizaciones. El updater intenta contactar el servidor aunque SUEnableAutomaticChecks esté en false. Fix sencillo pero visible — bloquea la primera impresión.

Impacto

  • Primera impresión al abrir la app
  • Fix rápido (1–2 líneas)

Complejidad

  • Baja
2

Bug: Right Command — primera pulsación no anda

El event tap de CGEvent se registra correctamente, pero la primera vez que se presiona Right ⌘ después de lanzar la app no dispara la grabación. Probablemente hay una race condition o el sistema tarda en activar el tap.

Impacto

  • Frustrante para usuarios nuevos
  • Core feature de la app

Complejidad

  • Media — necesita debug del event tap
3

Feature: UI de gestión de prompts

Hoy Settings solo muestra un Picker para elegir el prompt activo. No hay forma de crear, editar ni borrar prompts personalizados desde la app. El PromptStore ya existe — solo falta la UI.

Impacto

  • Alto — Refine Mode es el diferenciador
  • Sin UI, el usuario no puede personalizar

Complejidad

  • Media — CRUD con SwiftUI
4

Diseño: Pulir la interfaz

Settings, overlay de grabación, historial y ventana de first-run tienen UI funcional pero genérica. El objetivo es que se vea como una app terminada y pulida, lista para open source.

Impacto

  • Alto para open source / primera impresión
  • Diferencia una tool de un producto

Complejidad

  • Alta — muchas vistas a mejorar

Podés seleccionar varias. Los bugs (1 y 2) los hacemos nosotros; para UI (4) podemos definir exactamente qué pedirle a Claude Design.
